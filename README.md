## üöÄ Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code with Java extension, etc.)

### Running the Code
1. Clone or download this repository
2. Open the desired `.java` file
3. Compile: `javac filename.java`
4. Run: `java filename`
